# JALVORO Icon System — Clean Design Standard

Status: isolated design system; not connected to the existing workspace UI.

## Rules

1. One immediately recognizable object or concept.
2. Maximum two directly related objects when necessary.
3. Clean outer silhouette before internal detail.
4. No default wave, zigzag, sparkle, or decorative inner stroke.
5. Rounded line caps and joins.
6. Thin, balanced stroke with small-size readability.
7. `currentColor`; no embedded product colors.
8. No generic fallback icon for a specific category.
9. Inspect every symbol at 16, 20, 24, and 32 pixels.
10. No workspace-wide replacement without explicit visual approval.

Wave, zigzag, and subtle accents are experimental preview options only. Production defaults to `none`.

Existing screens continue to use their established icon source. Package aliases, compatibility fallbacks, and repository-wide icon codemods are prohibited without explicit approval.
