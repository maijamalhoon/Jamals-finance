# JALVORO brand assets

This directory is the replaceable public asset contract for the platform brand.

- `logo-mark.svg`: unique standalone JALVORO symbol.
- `wordmark.svg`: JALVORO wordmark.
- `app-icon.svg`: square application icon derived from the custom icon system.

The current production icon remains in place until the final custom JALVORO mark is approved. Replacing these assets must not require changes to application logic, database identifiers, native package IDs, or user data.

Canonical style source: `brand/brand.config.ts`.
